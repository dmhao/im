
**简要描述：**

- 设置管理员

**请求URL：**
- ` ip:port/v1/groups/:groupId/managers?appId=:appId `

**请求方式：**
- POST

**URI参数：**

|参数名|必选|类型|说明|
|:----    |:---|:----- |-----   |
|appId |是  |string/int |应用id   |

**POST参数：**

|参数名|必选|类型|说明|
|:----    |:---|:----- |-----   |
|ManagerId |是  |int |要被设为管理员的userId   |


 **返回示例**

 ```
 {
     "Code" : "0",
     "Msg" : "成功",
     "Data" : null
 }

 ```
